# MarvelApp

This page is a personal project to learn about Vue.js and a11y.

### We're using:

- Vue.js
- Vue-cli
- Sass
- [Marvel API](https://developer.marvel.com/)

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
